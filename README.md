# MineCloud

> A modern, extensible self-hosted cloud platform built with React, Spring Boot, and a microservice architecture.

MineCloud is more than just a private cloud drive.

It aims to provide a complete self-hosted personal cloud platform with secure file storage, intelligent file management, online collaboration, AI-powered capabilities, and an extensible plugin ecosystem.

> 🚧 This project is currently under active development.
